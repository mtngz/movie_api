myFlix Project: Marvelix

Create a REST API with Node.js

Objective

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request URL Endpoint HTTP Method Request body data format Response body data format
Get a list of all movies /movies GET None A JSON object holding data about all movies
Get data about a single movie by title; use %20 for spaces /movies/[title] GET None A JSON object holding data about a single movie, containing title, description, phase, director, image URL.

Example:
{
title: "Iron Man",
moviedescription: "Iron Man is a 2008 American superhero film based on the Marvel Comics character.",
director: {
dirname: "John Favreau",
bio: "Jonathan Favreau is an American actor, director, producer, and screenwriter.",
born: "1966",
died: ""
},
phase: {
number: "one",
phasedescription: "Phase One of the Marvel Cinematic Universe (MCU) is a series of American superhero films produced by Marvel Studios based on characters that appear in publications by Marvel Comics. The phase began in 2008 with the release of Iron Man and concluded in 2012 with the release of Marvel's The Avengers."
},
"imagepath": "ironman.png",
"featured": true
}
Get data about the phase by title; use %20 for spaces /movies/phase/[phase] GET None A JSON object holding data about the pase, containing number, phasedescription.

Example:
{
phase: {
number: "one",
phasedescription: "Phase One of the Marvel Cinematic Universe (MCU) is a series of American superhero films produced by Marvel Studios based on characters that appear in publications by Marvel Comics. The phase began in 2008 with the release of Iron Man and concluded in 2012 with the release of Marvel's The Avengers."
}
}
Get data about a director by name; use %20 for spaces /movies/directors/[name] GET None A JSON object holding data about a director, containing name, bio, birth year, death year.

Example:
{
director: {
dirname: "John Favreau",
bio: "Jonathan Favreau is an American actor, director, producer, and screenwriter.",
born: "1966",
died: ""
}
}
Add new user /users POST A JSON object holding data about the user to add, structured like:
{
username: "dummy",
email: "dummy@test.com",
password: "test"
}		 A JSON object holding data about the user that was added, including an userid and favorites.

Example:
{
username: "dummy",
email: "dummy@test.com",
password: "test",
userid: "1",
favorites: [],
}
Update the user info (username) /users/[userid]/[username] PUT None A text message indicating the username of a user with a userid was updated.
Add a movie to the user's favorites /users/[userid]/favorites/[title] PUT None A text message indicating the movie was added to the favorites of the user.
Delete a movie from the user's favorites /users/[userid]/favorites/[title] DELETE None A text message indicating the movie was deleted from the favorites of the user.
Delete a from the database /users/[userid] DELETE None A text message indicating the user was deleted from the database.
